## 경험 및 배움&#x20;

### JPA 활용2 강의 학습&#x20;

오늘은 조회 API 중에서, **여러 엔티티를 함께 조회하는 API**를 학습해보았다.

방법은 여러 가지가 존재했다.

1. 엔티티를 직접 노출시키는 방법
2. 엔티티를 DTO로 변환하는 방법
3. 엔티티를 DTO로 변환하고, 페치 조인을 사용하는 방법
4. JPA에서 DTO로 바로 조회하는 방법

> **결론 : 우선 엔티티를 DTO로 변환하는 방법을 시도하고, 성능 개선이 필요하다면 엔티티를 DTO로 변환하고, 페치 조인을 사용하는 방법을 사용해라!**
